Things to keep in mind throughout this work

  • Remeber to keep a growth mindset: When you're stuck, try using the word "yet." For example, think or say "I don’t understand how to call a function yet.
  • Be willing to make mistakes, they are going to happen
  • View "Problems" as Opportunities

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Within the main body of the html document you need bullet points and these are as follows
  • List Item
  • ul-unordered list
  • ol-ordered lists
  • There are also header tags that are needed in the body of an html document
  • Headers are numbers 1-6 with h going before the number

CSS

  • A Margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • "console.log" is used to test the interactivity of your website. This is done in the console of a page after right clicking 'inspect'

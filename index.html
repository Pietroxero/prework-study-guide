HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Commonly used elements to add content to pages: Text, Image, Header, List Elements
  • In addition to elements that can be seen on a page, elements are also used to divide the content on the page and to describe the content to browsers. Common elemet types that are used to divide and describe are: Meta-related, Content division, Semantic Elements
  • Each HTML element performs a unique role. However, two elements are especially important to understanding how the structure works: the head and body elements
    • - The head element contains information, or metadata, that helps the browser to render the page correctly. Elements included within the head are not visible when a user looks at a webpage
    - The body element contains the content that is shown to the user. All the elements that we want the user to see and interact with must be placed inside this element.
  • List Items:
    • - ul-unordered list
    - ol-Ordered lists
  • There are also header tags that are needed in the body of an html document.
  • Headers are numbers 1-6 with H going before the number.
  • Metadata:
    - Data about or webpage that the browser uses to effectively serve our webpage to users.
    - Typically not seen by users, but i contains important information about accessibility, search engine, and performance.
  • Inside the head element, we can use the following metadata tags:
    • Style element defines the CSS styles associated with a website.
    Base element defines the base URL for a webpage.
    Link element connects an external resource to the HTML document.
    Script element embeds client-side scripts in the HTML document.

CSS

  • A Margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • When a developer wants to style an element directly in the HTML file, they use inline CSS. Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • Inline CSS and internal style sheets are totally acceptable methods under certain conditions, but the most common way to style a webpage is by using an external CSS style sheet.
  • a selector div defines the element or attributes to which the rules, or declarations, will apply

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • "console.log" is used to test the interactivity of your website. This is done in the console of a page after right clicking 'inspect'
